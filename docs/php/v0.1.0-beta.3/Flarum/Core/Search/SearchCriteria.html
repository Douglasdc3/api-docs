SearchCriteria  

class SearchCriteria

Represents the criteria that will determine the entire result set of a search. The limit and offset are not included because they only determine which part of the entire result set will be returned.

Properties

User $actor The user performing the search.
string $query The search query.
array $sort An array of sort-order pairs, where the column is the key, and the order is the value. The order may be 'asc', 'desc', or an array of IDs to order by.
